Cómo trabajo

El stack detrás de cada caso

Diseño

Editorial-first. Fraunces / Inter, paletas warm y dark cinematic. Sin templates — cada marca recibe su sistema.

Construcción

HTML5 + CSS moderno + JS vanilla. Sin build steps, Lighthouse ≥ 90, Schema.org y SEO básico de fábrica.

Despliegue

GitHub Pages para demos, dominio propio para clientes. Pixel + GA4 + tracking server-side cuando aplica.

Captación

Meta Ads + GBP + chatbots WhatsApp con IA (Claude Haiku). Embudo medido punta a punta.
